# TC Helper

A Credit Card expense tracker and management system that syncs with Gmail.

## Tech Stack
- Frontend: React (Vite)
- Backend: Python (FastAPI)
- Database: PostgreSQL (Railway) / SQLite (Local)
- Deployment: Docker (Railway)

## Development

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
